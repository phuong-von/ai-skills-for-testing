# ai-review-automation

Reviews a Playwright automation test script against coding, Playwright, and locator standards before merge.

Self-contained — bundles its own reference knowledge base, so it installs and works standalone without any other project setup.

## Install

Download [`ai-review-automation.skill`](./ai-review-automation.skill) and install it in Cowork via **Settings > Capabilities > Skills > Install from file**.

See `SKILL.md` for the full specification (inputs, outputs, execution rules, self-validation).
