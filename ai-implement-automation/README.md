# ai-implement-automation

Converts an approved test case into a Playwright automation script that consumes existing Page Objects.

Self-contained — bundles its own reference knowledge base, so it installs and works standalone without any other project setup.

## Install

Download [`ai-implement-automation.skill`](./ai-implement-automation.skill) and install it in Cowork via **Settings > Capabilities > Skills > Install from file**.

See `SKILL.md` for the full specification (inputs, outputs, execution rules, self-validation).
