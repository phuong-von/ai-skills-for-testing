# ai-review-test-case

Reviews test cases for coverage, clarity, and traceability before they're approved for automation or manual execution.

Self-contained — bundles its own reference knowledge base, so it installs and works standalone without any other project setup.

## Install

Download [`ai-review-test-case.skill`](./ai-review-test-case.skill) and install it in Cowork via **Settings > Capabilities > Skills > Install from file**.

See `SKILL.md` for the full specification (inputs, outputs, execution rules, self-validation).
