# ai-review-page-object

Reviews a Playwright Page Object class against POM and locator standards before merge.

Self-contained — bundles its own reference knowledge base, so it installs and works standalone without any other project setup.

## Install

Download [`ai-review-page-object.skill`](./ai-review-page-object.skill) and install it in Cowork via **Settings > Capabilities > Skills > Install from file**.

See `SKILL.md` for the full specification (inputs, outputs, execution rules, self-validation).
