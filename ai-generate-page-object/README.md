# ai-generate-page-object

Generates a Playwright Page Object class for a page or UI section, following POM and locator priority standards.

Self-contained — bundles its own reference knowledge base, so it installs and works standalone without any other project setup.

## Install

Download [`ai-generate-page-object.skill`](./ai-generate-page-object.skill) and install it in Cowork via **Settings > Capabilities > Skills > Install from file**.

See `SKILL.md` for the full specification (inputs, outputs, execution rules, self-validation).
