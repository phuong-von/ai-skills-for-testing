# ai-analyze-requirements

Analyzes user stories and requirements for completeness and testability, producing a Ready/Not-Ready analysis report with clarification questions.

Self-contained — bundles its own reference knowledge base, so it installs and works standalone without any other project setup.

## Install

Download [`ai-analyze-requirements.skill`](./ai-analyze-requirements.skill) and install it in Cowork via **Settings > Capabilities > Skills > Install from file**.

See `SKILL.md` for the full specification (inputs, outputs, execution rules, self-validation).
