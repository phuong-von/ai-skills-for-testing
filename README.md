# AI Skills for Testing

A collection of installable Cowork skills for QA/testing workflows.

## Skills in this repo so far

| Skill | Purpose |
|---|---|
| [`ai-design-test-cases`](./ai-design-test-cases) | Generate test cases from an analyzed requirement using approved test design techniques. |

More skills (`ai-analyze-requirements`, `ai-generate-page-object`, `ai-implement-automation`, `ai-review-test-case`, `ai-review-automation`, `ai-review-page-object`, `qc-onboarding-guide`, `qc-setup-guide`, `review-playwright-auto`) will be added in future commits.

## Install

Download the `<skill-name>.skill` file from the skill's folder and install it in Cowork via **Settings > Capabilities > Skills > Install from file**.
